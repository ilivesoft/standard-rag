# Codemaps Overview: Local RAG Pipeline

> 이 문서는 신규 프로젝트 초기화 시 생성된 아키텍처 플레이스홀더입니다.
> 코드 구현 후 `/moai:codemaps` 명령어로 실제 코드 분석 기반 문서로 업데이트하세요.

## 설계 원칙

- **모듈성**: 각 파이프라인 단계가 독립 모듈로 교체 가능
- **로컬 우선**: 외부 API 없이 완전 로컬 실행 지원
- **유연성**: 임베딩 모델, LLM, Vector DB를 설정으로 교체 가능
- **테스트 가능성**: 각 모듈 단위 테스트 독립 실행 가능

---

## 주요 설계 결정

| 결정 | 선택 | 대안 | 이유 |
|------|------|------|------|
| Vector DB | ChromaDB | FAISS, Qdrant | 로컬 영속 저장, 설치 간편 |
| 임베딩 | BAAI/bge-m3 | OpenAI ada-002 | 한국어 지원, 무료, 로컬 실행 |
| 하이브리드 검색 | Vector + BM25 | Vector만 | 키워드 기반 쿼리 정확도 향상 |
| Demo UI | Gradio | Streamlit | 파일 업로드 컴포넌트 내장, 빠른 구성 |
| LLM (로컬) | Ollama | llama.cpp | API 서버 형태, Python 연동 용이 |
